Welcome to the GraphQLBuilder documentation!
===================================

**GraphQLBuilder** is a Python library that helps you to build GraphQL queries in your Python code.
Its usage is mainly focused on `Hasura.io <https://hasura.io/>`_ endpoints, but it can be used with any GraphQL endpoint.

.. note::

   This project is under active development.
